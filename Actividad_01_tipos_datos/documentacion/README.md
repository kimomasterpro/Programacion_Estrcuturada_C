# Actividad 01: Tipos de Datos en C

## Descripción
Este programa realiza la declaración, inicialización y visualización de los tipos de datos comunes y primitivos en el lenguaje C. El objetivo principal es mostrar de forma tabulada el valor almacenado por cada tipo de dato junto con su dirección de memoria real utilizando punteros (`%p`).

## Temas aplicados
* Tipos de datos básicos (enteros, flotantes, caracteres y booleanos)
* Especificadores de formato para `printf` (`%d`, `%u`, `%f`, `%Lf`, `%c`, `%s`, `%p`)
* Operador de dirección (`&`) y conversión a apuntador genérico `(void*)`
* Formateo y alineación de texto en consola
