# Actividad 03: Estructuras Selectivas Anidadas

## Descripción
Este programa captura un número entero proporcionado por el usuario y realiza una doble validación mediante sentencias condicionales:
1. Comprueba si el número se encuentra en un rango cerrado válido (mayor a 1 y menor o igual a 10).
2. Determina mediante comparaciones directas si el número pertenece al conjunto de los números primos dentro de dicho intervalo.

## Temas aplicados
- Estructuras condicionales selectivas (`if` / `else`)
- Sentencias de control anidadas
- Operadores lógicos de conjunción (`&&`) y disyunción (`||`)
- Operadores relacionales de comparación (`>`, `<=`, `==`)
